Atarashii by BonsaiDen

Atarashii(Japanese for "New") is a Twitter Client specifically designed for the GNOME Desktop. It uses GTK+ and webkit to archive a slim and functional design. The code itself is written in Python and uses the corresponding Python bindings for the GTK+ libraries.

Atarashii aims to provide a simple and good Twitter experience on your GNOME desktop. Main features a timeline that includes mentions, support for direct messages and notifications about new Tweets directly on your desktop.

Dependencies

python (>= 2.6) python-webkit python-gtk2 python-gconf python-gobject python-dbus python-notify mplayer-nogui

Install

In order to install Atarashii first download the source and extract it. After that cd to the folder you've extracted the source to and run "python make.py". Then just install the created Debian-Package.
